# Employee and Department SQL Analysis

This is a beginner-friendly SQL mini project based on employee and department data.  
The project is created to practice important SQL concepts used in data analytics.

## Project Objective

The main objective of this project is to analyze employee and department records using SQL queries.  
This project helps in understanding how to retrieve, filter, sort, group, and combine data from multiple tables.

## Topics Covered

This project uses the following SQL concepts:

- SELECT
- WHERE
- ORDER BY
- GROUP BY
- DISTINCT
- JOIN
- LEFT JOIN
- Aggregate Functions (`COUNT`, `SUM`, `AVG`, `MAX`)

## Tables Used

### 1. employees
This table stores employee details such as:
- Employee ID
- Employee Name
- Department
- Salary
- City
- Join Date

### 2. departments
This table stores department details such as:
- Department ID
- Department Name
- Manager Name

## Project Tasks

The project includes queries for:

- Viewing all employee records
- Filtering employees by department, salary, city, and join date
- Sorting employees by salary and joining date
- Counting employees department-wise and city-wise
- Calculating average, total, and highest salary by department
- Joining employee and department tables
- Finding departments with no employees
- Finding employees whose department details are missing

## Files in this Repository

- `employee_department_project.sql` → contains table creation, insert statements, and SQL queries

## Sample SQL Queries Included

- Show all employees

- Find average salary in each department

- Find departments with no employees

After completing this project, I learned how to:
